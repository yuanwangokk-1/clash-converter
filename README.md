# Clash订阅链接转换工具

一个在线的Clash订阅链接转换工具，支持单链接转换、批量处理、参数自定义和配置管理。

## 功能特性

 🚀 主要功能
- 单链接转换：快速转换单个订阅链接
- 批量转换：同时处理多个订阅链接
- 参数自定义：可配置Emoji、UDP、TCP快速打开等参数
- 配置管理：支持配置模板选择和历史记录管理

 ⚙️ 配置模板
- 默认模板：ACL4SSR Online (基础规则)
- 极简模板：ACL4SSR Mini (精简规则)
- 完整模板：ACL4SSR Full (完整规则)
- 自定义模板：支持自定义配置URL

 📊 其他特性
- 历史记录：自动保存转换记录
- 导入导出：支持配置文件导入导出
- 实时日志：详细的操作日志记录
- 响应式设计：完美适配桌面和移动设备
- 本地存储：配置和历史记录保存在本地
